Define one or more build parameters to inject into every branch job. All standard Jenkins parameter types are supported:

  • String Parameter – a free-text value
  • Boolean Parameter – a true/false checkbox
  • Choice Parameter – a drop-down list of fixed values
  • Password Parameter – a masked string value
  • Multi-line String Parameter – a text area

Third-party parameter plugins (e.g. Extended Choice Parameter, Git Parameter) are also available if installed.
