Controls how UI-defined parameters interact with a parameters {} block declared inside the branch's Jenkinsfile.

Unchecked (default)
Parameters defined here replace any parameters declared in the Jenkinsfile. The Jenkinsfile parameters are completely ignored.
Checked
Parameters are merged. Jenkinsfile parameters are kept as a base, and parameters defined here are overlaid on top. If the same parameter name appears in both places, the value defined here wins.
